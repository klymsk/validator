# validator

## 📈 Опис лабораторної роботи
#### Метою даної роботи є проектування консольної утиліти для перевірки відповідності структурованих даних заданій схемі. Система являє собою CLI-програму.

## 📝 Опис проекту
#### Проект є Python інструментом, котрий функціонує для перевірки складних вкладених структур даних (наприклад файли JSON), на відповідність представленого шаблону.

#### Сам алгоритм є наступним:

- зчитує файл схеми
- зчитує файл даних
- будує внутрішню модель типів
- виконує перевірку даних
- формує звіт про помилки
Програма перевіряє вкладені структури даних та підтримує рекурсивні типи.

## 📥 Інсталяція

1.  **Клонуємо репозиторій:**
    ```bash
    git clone git clone https://github.com/klymsk/validator.git

    cd validator
    ```

2.  **Віртуальне оточення venv:**

    *Встановлення:*
    ```bash
    python3 -m venv venv
    ```

    *Активація:*
    *Mac/Linux:*
    ```bash
    source venv/bin/activate
    ```

    *Windows:*
    ```bash
    venv\Scripts\activate
    ```

3. **Встановити залежності:**
    ```
    pip install -r requirements.txt
    ```

## 🔍 Linter

### Використовую Flake8 для статичного аналізу:

*Запуск лінтера:*
    ```bash
    flake8 .
    ```

### Після, виконується перевірка:

- Стилю написання коду
- Синтаксису
- Відповідність кодуванню PEP8
